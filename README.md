# Landing Page - Energia Solar

Este projeto é uma landing page para captação de clientes de energia solar, desenvolvida com React, TypeScript, Vite e shadcn/ui.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Como rodar localmente

```bash
pnpm install
pnpm dev
```
